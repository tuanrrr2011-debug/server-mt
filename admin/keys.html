About License Keys

Format: License keys are generated in the format XXXX-XXXX-XXXX

Status:

  • Available: Key is not yet bound to any device
  • Bound: Key is bound to a device and ready to use

Actions:

  • Copy: Copy the key to clipboard
  • Reset: Remove device binding (make available again)
  • Delete: Permanently delete the key
